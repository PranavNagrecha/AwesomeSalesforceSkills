# Gotchas — Pull Request Policy Templates

## Gotcha 1: CODEOWNERS without team

**What happens:** User leaves; PRs stuck.

**When it occurs:** Individual username listed.

**How to avoid:** Use GitHub teams.


---

## Gotcha 2: Template too long

**What happens:** Authors delete it.

**When it occurs:** 15-section template.

**How to avoid:** ≤5 sections, one screen.


---

## Gotcha 3: Coverage check both warn+block inconsistent

**What happens:** Confusion.

**When it occurs:** Two overlapping checks.

**How to avoid:** One authoritative coverage check.
